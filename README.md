# 💰 Smart Expense Tracker

A professional Python-based CLI application for tracking, managing, and analyzing expenses using Data Structures & Algorithms concepts.

---

# 🚀 Features

- ➕ Add Expenses
- 📋 View Expenses
- 🔍 Search Expenses
- 📂 Search by Category
- 📈 Sort Highest Spending
- 📉 Sort Lowest Spending
- 📊 Monthly Summary
- 💸 Highest Expense Analysis
- 🏆 Highest Spending Category
- 📅 Weekly Spending Analytics
- 🕒 Recent Transactions
- 🤖 Smart Spending Insights
- 💾 JSON Data Storage
- 🎨 Professional Terminal UI

---

# 🛠️ Technologies Used

- Python
- JSON
- Colorama
